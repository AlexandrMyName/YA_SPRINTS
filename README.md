# 🚀 YA Sprint One – Event Management API

RESTful API для управления событиями. Проект выполнен в рамках учебного спринта. Реализованы базовые CRUD-операции, валидация, версионирование, Swagger-документация, in-memory репозиторий  

## 📦 Технологии
- **.NET 8** (или ваша версия)
- **ASP.NET Core Web API**
- **Swagger / Swashbuckle** – автоматическая документация
- **AutoMapper** – маппинг бизнес сущностей и DTO
- **FluentValidation / DataAnnotations** – валидация (укажите что используете)
- **In‑memory хранение** (`ConcurrentDictionary`)
- **API Versioning** (1.0)
- **CORS** – настроены политики доступа
